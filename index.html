📥 Import DPGF
Chargez votre DPGF Excel. L'application détecte automatiquement les feuilles contenant des colonnes Repère / Descriptif / Unité / Quantité. Pour chaque ligne, l'app proposera automatiquement les repères les plus probables grâce à l'analyse de similarité textuelle.
Lignes DPGF Affichage :
💡 Astuce : Tapez un prix directement, basculez entre Exact / Ratio, ou cliquez sur Choisir pour piocher dans la base. Les ⭐ indiquent les repères suggérés automatiquement.
📋 Saisie masse — Ajout de plusieurs prix d'un coup
Saisissez vos prix directement dans le tableau ci-dessous, ligne par ligne. Vous pouvez aussi coller depuis Excel/Calc (Ctrl+C puis Ctrl+V dans la zone), ou utiliser une trame Excel téléchargeable.
🗂️ Trame Excel :
Aperçu
#Repère*Lot*Désignation* UnitéPrix HT*DateProjet
💡 Les lignes vertes sont valides. Les rouges ont des champs manquants et seront ignorées. ⚠️ jaune = prix anormal (très en-dehors du min-max historique du repère).
📊 Ratios par repère interne
Pour chaque repère interne, l'application calcule la moyenne (ratio), le min, max et le dernier prix observé. Cliquez sur une ligne pour voir le détail des prix.
📜 Historique des sauvegardes automatiques
L'application crée automatiquement des sauvegardes à chaque opération importante (sync Excel, import masse/DPGF, suppressions, reset). Les 20 dernières sont conservées dans .
